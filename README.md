## About The Project

[![Product Name Screen Shot][product-screenshot]](https://example.com)

The **Clinical Note Simplifier** is a lightweight, fast, and user‑friendly tool designed to convert complex clinical notes into patient‑friendly language. It uses **Groq’s ultra‑fast LLM**, computes readability metrics, and presents results in a clean, medical‑grade UI.

Features include:

- Clinical note simplification  
- Readability scoring (Flesch, SMOG, Gunning Fog, etc.)  
- Copy‑to‑clipboard  
- Download as PDF  
- Loading spinner + fade‑in animations  
- Clean, modern interface

## Getting Started

Follow these steps to run the project locally.

### Prerequisites

Install Python 3.10+ and pip.

### Installation

1. Clone the repo  
   ```sh
   git clone https://github.com/sanskritim05/clinical-note-simplifier.git
   python -m venv venv 
   source venv/bin/activate # macOS/Linux 
   venv\Scripts\activate # Windows
   ```
